{% translate "Three SIM Provider Public Keys" %}

{% translate "The PGP public keys for the SIM provider are required to encrypt the 3 XOR key shares generated for each wireless keys. The provider will reconstruct each key after decypting the shares with their private keys." %}

{% translate "Five Security Officer Public Keys" %}

{% translate "A unique 128-bit wrap key is generated for each wireless key. The wrap key is used to wrap/encrypt its associated wireless key. Each wrap key is then split into 5 shares using " %}{% translate "Shamir Secret Splitting (SSS)" %}{% translate ". Each of these shares will be encrypted with a PGP public key belonging to a security officer. Three security officers will be required to reconstruct this wrap key which in turn can be used to decrypt its wireless key." %}

{% translate "Two Yubikey Public Keys" %}

{% translate "These are the backup/fallback keys that will be used to protect/encrypt the wireless keys. In the event that the number of available Security Officers do not meet the threshold to decrypt the key, this will be used for decryption. Each Yubikey and its encrypted artifacts will be stored in a safety deposit box in two different locations." %}

{% translate "Two Public Keys for Milenage" %}

{% translate "The milenage key will be entered at the terminal of the network provider. To mitigate the risk of both milenage keys been known by a single resource, two public keys belonging to two different resources will be provided. One to encrypt the milenage Transport key and the other to encrypt the milenage OP key." %}
