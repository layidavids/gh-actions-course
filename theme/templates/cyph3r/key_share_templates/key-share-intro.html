{% translate "Key Sharing" %}

{% translate "Key sharing or splitting, also known as secret sharing, refers to a cryptographic technique where a secret (such as a cryptographic key) is divided into multiple parts, or shares, and distributed among participants." %}

{% translate "The key idea behind this process is that no single share reveals enough information to reconstruct the original secret. This app supports the following schemes for key sharing:" %}

{% translate "Shamir's Secrets Sharing" %}

{% translate "This app only supports splitting" %}128 bit{% translate " keys using Shamir's secret sharing. The key can be broken into n shares and a minimum of m shares (m < n) otherwise known as the threshold is required for key reconstruction." %}

{% translate "Xor-based Key Splitting" %}

{% translate "A much simpler approach than Shamir is using the XOR or bitwise operation to split a secret. All the shares are required to reconstruct the secret." %}

{% translate "The assumption is the key shares are also expected to remain secret. Consequently, every key splitting operation requires PGP public files to be uploaded. The number of files to be uploaded must be the same number as that of the shares to be created." %}

{% translate "Key reconstruction operations require only a single PGP file to be uploaded." %}

{% translate "This app only accepts PGP public key files that are in the " %}{% translate "ASCII armored format" %}{% translate " i.e. the contents of each public key file must have the begin and end headers of" %} ----- BEGIN PGP PUBLIC KEY BLOCK----- and -----END PGP PUBLIC KEY BLOCK-----{% translate " respectively." %}
